# Customer Churn Analysis Using SQL

## Project Overview
Analyzed customer churn using SQL by performing data exploration, churn metric calculations, cohort analysis, and business insight generation.

## Objectives
- Explore customer data
- Calculate churn metrics
- Perform cohort analysis
- Identify churn drivers

## Tools & Technologies

## Key Insights
- Identified high-risk customer groups
- Analyzed churn by tenure and contract type
- Evaluated customer lifetime value
- Generated actionable retention strategies

I performed churn analysis on telecom data using SQL. I calculated churn rate, 
segmented customers by tenure and contract type, identified high-risk customers,
and estimated revenue loss. This helped in understanding key factors driving customer churn.
